## Requirements
### Requirement: Time Since Last Event
The system SHALL display the time elapsed since the last increment or decrement event.

#### Scenario: Displaying time since
- **WHEN** a counter row is displayed
- **THEN** it SHALL show text like "Last: 2 hours ago" or "Last: 5 minutes ago" based on the latest history event
